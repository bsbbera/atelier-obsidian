# Dividers

A plain `---` horizontal rule is restyled as a fine copper seam. For ornament, use the `hr` callout casts.

| Cast | Ornament |
|------|----------|
| `ros` | Rosette — centred medallion |
| `fl` | Fleuron — leaf flourish |
| `str` | Stratum — stratigraphic seam |

> [!hr|ros]

> [!hr|fl]

> [!hr|str]

## Syntax

```markdown
Plain seam:

---

Ornamented:

> [!hr|ros]
```

**Use it for:** `ros`/`fl` → section breaks in long-form prose where a plain rule feels too plain. `str` → chapter / phase boundaries. Plain `---` is fine everywhere else.
